# Sales Dashboard & Customer Insights

## Project Objective

Build an interactive sales dashboard and generate actionable customer insights from sales data.

## Features

* Sales Analysis
* Customer Insights
* Interactive Dashboard
* Visualization Reports
* Business Recommendations

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Streamlit
* GitHub

## Key Insights

* Technology category generated the highest sales.
* Top customers contributed a significant share of revenue.
* Western region showed the highest profitability.
* Consumer segment accounted for the largest customer base.

## Business Recommendations

1. Focus marketing efforts on top customers.
2. Expand inventory in high-performing regions.
3. Improve profit margins in low-profit categories.
4. Implement customer loyalty programs.
